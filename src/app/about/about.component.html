About this page AngularCV: a simple self-hosted online-CV

This website was created as a GitHub project for the purpose of having a basic online-CV, which anyone can host by themselves. The main target audience are software developers though. To build this, basically the following technologies are used:

AngularJS 4 Material Design GitHub CSS TypeScript JetBrains WebStorm AWS
